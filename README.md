# PhiOS

PhiOS is a simple CLI 16-bit operating system.

## How to Run

1. Install QEMU:
   ```bash
   sudo apt install qemu
2. Use the Makefile to build
    ```bash
    make
3. Run build.sh
   ```bash
   sh build.sh
